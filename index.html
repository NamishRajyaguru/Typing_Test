Welcome to the Typing Speed Test! My name is Namish, and I built this project as a way to learn and practice the basics of JavaScript. I’ve been working with JavaScript for a short time, but with the help of ChatGPT and many online resources, I was able to put this project together. This project was made as a fun way to test typing speed and accuracy while improving my programming skills.


The project works by allowing users to type a sentence that appears on the screen. Once the user starts typing, the application tracks their performance, showing the characters they've typed, and highlights correct and incorrect letters in real time. A timer starts when the user types their first character and stops when the sentence is completed. After completing the test, users can save their results, which include their typing speed (measured in words per minute) and accuracy. The results are then added to a leaderboard, allowing users to compare their performance with others.


The app uses JavaScript to dynamically update the content of the webpage, ensuring an interactive experience for users. By capturing keypress events, the app registers each letter typed and highlights the letters based on whether they are correct or incorrect. Additionally, the app displays useful information such as the total number of characters typed, the time taken, and the final typing speed and accuracy.


As a beginner, I’ve gained valuable experience through this project, learning how to work with DOM manipulation, event handling, local storage, and creating simple interactive applications. I hope to continue improving my coding skills with more complex projects in the future.
